npmjs.org - пакеты для node js
npm init - инициализация npm

npm uninstall @babel/core - удаление  пакета
npm i @babel/core - установка пакета
npm i -g @babel/core - флаг "-g" если нужно установить глобально

npm i -g @babel/core --save - для будущего продакшена(которые участвуют в продакшене)
npm i -g @babel/core --save-dev - пакет участвует только в разработке

npm i - инициализация node с учетом существующего package.json

webpack - сборщик для интеграции множества отдельных частей приложения в единое целое и работы со вспомогательными инструментами. 

babel - для перевода js с современных стандартов в более старых. 


npm run build - запуск постройки билда
